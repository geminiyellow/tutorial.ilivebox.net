#5人でジャンケンを行い、勝者1人を選ぶ

  - ルール
    - 全て同じなら引き分け
    - グ−、チョキ、パー 全てが出たら引き分け
    - 負けた人その後の勝負には参加出来ない
  - できれば5人限定ではなくn人で出来るようにする
  - ファイル名 jyanken5.rb
  - 実行例

  
    % ruby jyanken5.rb
    1: グー  2: チョキ  3: パー  4: グー  5: パー  
      引き分け
    1: チョキ  2: パー  3: チョキ  4: パー  5: グー  
      引き分け
    1: チョキ  2: パー  3: パー  4: グー  5: パー  
      引き分け
    1: グー  2: グー  3: グー  4: グー  5: チョキ  
      勝ち: グー
    1: チョキ  2: パー  3: パー  4: グー  5: x  
      引き分け
    1: チョキ  2: チョキ  3: チョキ  4: パー  5: x  
      勝ち: チョキ
    1: グー  2: グー  3: パー  4: x  5: x  
      勝ち: パー
    
    勝者: 3
